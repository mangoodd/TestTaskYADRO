

csvreader.exe - программа для чтения произвольной CSV-формы, написанная на языке Python 3 с использованием стандартных библиотек.
Программа 'csvreader' вычисляет значения ячеек, если это необходимо и выводит получившуюся табличку в виде CSV-представления в консоль.



Запуск программы производиться через консоль. Пример запуска программы:

['путь до программы'] csvreader.exe file.csv

где csvreader.exe – имя исполняемого файла (может быть любым), file.csv – имя файла с таблицей.


Для тестирования в папке с программой находятся файл с расширением .cmd ('start_test.cmd'), который запускает тестовые сценарии работы программы.
Тестовые варианты таблиц находятся в папке CSVFiles.
